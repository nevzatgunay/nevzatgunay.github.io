## ng

It is a `npm` package and when you run in the terminal, you will see details of my portfolio.

  - [GitHub](https://github.com/nevzatgunay/ng)
  - [npm](https://www.npmjs.com/package/nevzatgunay)

## How to develop

 - Clone the project,
 - Navigate to the root folder of it, and type

```
node nevzatgunay.js
```

### How to use

  - In the terminal, type

```
npm i nevzatgunay
```

or

```
npx nevzatgunay
```
